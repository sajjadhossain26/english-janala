# 📚 English Janala

English Janala is a simple web application designed to help users learn English vocabulary easily. The project provides different vocabulary levels and displays words with pronunciation and interactive controls.

## 🚀 Features

* Multiple vocabulary levels
* Interactive vocabulary cards
* Word pronunciation display
* Clean and responsive UI using Tailwind CSS
* Simple and beginner-friendly JavaScript logic

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* JavaScript (ES6)
* Font Awesome Icons

## 📂 Project Structure

```
english-janala
│
├── index.html
├── script.js
├── style.css
├── tailwind.config.js
└── assets/
```

## ⚙️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/english-janala.git
```

2. Open the project folder

```
cd english-janala
```

3. Open **index.html** in your browser.

## 🎯 Project Goal

The goal of this project is to practice **JavaScript DOM manipulation, API handling, and responsive UI design using Tailwind CSS**.
